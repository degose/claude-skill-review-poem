# 🐢 review-poem

Generate a humorous Korean poem from a GitHub PR diff.  
Inspired by CodeRabbit-style PR poems, but with a calm and realistic "turtle developer" tone.

---

## ✨ Features

- Turns PR changes into a short Korean poem
- Reflects actual code changes (not random poetry)
- Captures developer pain, bugs, and refactors
- Light humor + realistic dev emotions
- Clean markdown output for PR comments

---

## 🚀 Usage

```bash
/review-poem 123
```
